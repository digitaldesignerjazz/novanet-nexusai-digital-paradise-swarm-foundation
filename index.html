THE VISION

A paradise that
grows itself.

We are building a hybrid physical-digital ecosystem where resilient mesh networks and intelligent agent swarms collaborate to create, maintain, and evolve a living Digital Paradise.

Starting from Hannover, Germany, NovaNet provides the decentralized communication fabric. Nexus AI forms the collective intelligence. Together they enable a sovereign, self-improving space where humans and agents co-create — with privacy, beauty, and autonomy at the core.

GET STARTED

Run the foundation in under a minute.

Clone the repo and launch the integrated NovaNet + Nexus AI demo stack.

Standalone Demo (Recommended first)
git clone https://github.com/digitaldesignerjazz/novanet-nexusai-digital-paradise-swarm-foundation.git
cd novanet-nexusai-digital-paradise-swarm-foundation

python3 examples/swarm_agent_demo.py
Full Docker Stack (NovaNet + Nexus AI)
cp .env.example .env
docker compose up --build

Launches both the mesh node placeholder and the full Nexus AI swarm demo.
